# 故障排查

## 双击后窗口闪退

请从完整解压后的目录运行。查看根目录的 `startup-error.log`。如果 `tools/python/` 只下载了一部分，删除该目录后重新启动。

## 手机打不开地址

- 局域网模式：确认手机和电脑在同一 Wi-Fi，且 Windows 防火墙允许专用网络。
- 外网模式：保持启动窗口开启，使用本次启动显示的新 HTTPS 完整地址。
- 不要省略网址中的 `?token=...`。

## 页面是旧版本

关闭旧服务，使用新地址重新打开；必要时清除该页面缓存。服务端已发送 `Cache-Control: no-store`。

## 鼠标能动但文字不能输入

先点击电脑端目标输入框，再使用全屏“文字输入”面板发送。管理员应用需要以管理员身份启动本程序才能控制。

## 电脑声音或麦克风失败

1. Windows 设置 → 隐私和安全性 → 麦克风，允许桌面应用访问。
2. 确认 Windows 已设置默认麦克风和默认扬声器。
3. 关闭占用独占音频设备的程序。
4. 重启服务后再试。

`data discontinuity in recording` 是 Windows 音频缓冲提示；当前版本使用持续采集缓冲，并会过滤该警告。

## 摄像头失败

允许桌面应用访问摄像头，并关闭其他独占摄像头的程序。当前版本使用默认摄像头（设备编号 0）。

## 手机麦克风无法开启

必须使用 HTTPS 外网地址或浏览器认可的安全来源，并允许网页使用麦克风。实时对讲建议佩戴耳机。

## 外网隧道创建超时

检查电脑能否访问 GitHub 和 Cloudflare。企业、校园或公共网络可能阻止隧道连接。

## 彻底重置运行环境

关闭程序后删除 `.venv/` 和 `tools/`，再重新双击启动文件。不要删除 `received_files/` 中仍需要的文件。
