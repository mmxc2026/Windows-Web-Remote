# 使用指南

## 首次启动

双击 `start-local.bat` 或 `start-internet.bat`。启动器会：

1. 查找可用的 Python 3。
2. 在项目目录创建独立的 `.venv`。
3. 安装屏幕、摄像头和声音依赖。
4. 启动控制服务并显示手机访问地址。

如果启动失败，错误会写入项目根目录的 `startup-error.log`。

## 触摸操作

| 操作 | 效果 |
| --- | --- |
| 轻触画面或触摸板 | 左键单击 |
| 连续轻触 | 双击 |
| 按住后移动 | 左键拖拽 |
| 长按不动 | 右键单击 |
| 触摸板单指滑动 | 移动鼠标 |
| 触摸板双指滑动 | 滚轮 |

## 全屏和文字

- 点击顶部“全屏”只保留电脑画面。
- “文字输入”会显示一个手机端输入框；输入后点击“发送”。
- “电脑键盘”通过 `Win+R → osk → Enter` 打开 Windows 屏幕键盘。
- 发送文字前，应先在远程画面中点击电脑的目标输入框。

## 声音和摄像头

- “电脑声音”播放 Windows 当前输出声音。
- “电脑麦克风”播放电脑默认麦克风。
- “电脑摄像头”打开默认摄像头；关闭摄像头页面后停止占用。
- “按住说话”只在按住时把手机麦克风传到电脑。
- “实时对讲”持续传输手机麦克风，再次点击停止。

手机麦克风要求 HTTPS 或浏览器认可的安全环境。实时双向对讲建议佩戴耳机，避免声学回路和啸叫。

Windows 设置中需要允许桌面应用访问摄像头和麦克风。

## 文件传输

### 手机传到电脑

在“媒体与文件”区域点击“上传到电脑”。文件保存在 `received_files/`，单个文件上限为 512 MB。

### 电脑传到手机

把文件复制到 `shared_files/`，然后在手机点击“刷新下载”并选择文件。

## 停止服务

关闭启动窗口或按 `Ctrl+C`。外网临时地址会立即失效，下次启动会生成新地址和新密钥。
